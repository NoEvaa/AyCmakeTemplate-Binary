# C++ Cmake Template: Binary

A cmake template for C++ binary library.

更多相关内容请移步[AyCmakeTemplate](https://gitee.com/ayin-tech/ay-cmake-template)。

## 适用场景

- 二进制形式静态库/动态库

## 快速开始

### 构建

```bash
sh build.sh
sh build.sh Release
```
